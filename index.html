2. Finalidad de la aplicación

Módulo Tutorías CASEI es una aplicación móvil desarrollada con fines académicos y de prueba, orientada a la visualización de información de tutorías, analítica académica y mecanismos de seguridad móvil.

Durante esta etapa, la aplicación utiliza datos ficticios o de prueba. No se utilizarán datos reales de alumnos.

3. Usuarios y testers

La aplicación está dirigida únicamente a testers autorizados. Todos los testers participantes son mayores de edad.

Las cuentas se proporcionan previamente a testers autorizados. Aunque la aplicación incluye una pantalla de registro, su uso está limitado a pruebas controladas y no está destinado al registro público libre.

4. Datos que puede tratar la aplicación

La aplicación puede tratar los siguientes datos:

  • Datos de cuenta: correo electrónico, nombre, apellidos, rol e identificador de usuario.
  • Datos académicos ficticios: perfiles académicos, métricas de segmentación, promedio, asistencia, cohorte y programa educativo.
  • Datos técnicos del dispositivo: token de Firebase Cloud Messaging, plataforma del dispositivo y fecha de última actividad.
  • Datos de seguridad: estado de sesión, detección de ubicación simulada, detección de entorno no seguro y cierre por inactividad.
  • Datos locales del baúl seguro: contraseña guardada, notas privadas, número de teléfono y datos de tarjeta, únicamente cuando el usuario decida capturarlos voluntariamente.

5. Uso de ubicación

La aplicación puede solicitar acceso a la ubicación del dispositivo únicamente con fines de seguridad, para detectar el uso de ubicaciones simuladas o Fake GPS.

La ubicación se revisa únicamente de forma local en el dispositivo. La aplicación no envía ni almacena coordenadas de ubicación en Supabase, Firebase ni en ningún otro servidor.

6. Baúl seguro

La aplicación incluye un baúl seguro donde el usuario puede guardar voluntariamente información confidencial local, como una contraseña, notas privadas, un número de teléfono y datos de tarjeta.

Esta información permanece únicamente en el teléfono mediante los mecanismos de almacenamiento seguro del dispositivo. No se envía a Supabase ni a Firebase.

El usuario puede eliminar manualmente la información almacenada en el baúl seguro desde la aplicación.

Además, en caso de pérdida, robo o riesgo de acceso no autorizado, el responsable podrá activar un mecanismo de borrado remoto para eliminar la información local protegida del dispositivo.

7. Firebase Cloud Messaging y borrado remoto

La aplicación utiliza Firebase Cloud Messaging para recibir mensajes técnicos de seguridad. Para ello, puede registrar un token de dispositivo asociado a la cuenta del usuario.

Este token se utiliza únicamente para fines técnicos y de seguridad, incluyendo el envío de una instrucción de borrado remoto cuando exista riesgo de pérdida, robo o acceso no autorizado al dispositivo.

El borrado remoto elimina información local protegida, como la sesión y los datos del baúl seguro. No elimina automáticamente datos institucionales o académicos almacenados en servidores.

8. Eliminación de cuenta y datos

Actualmente, la aplicación no cuenta con una opción interna para eliminar la cuenta. La eliminación de cuenta se gestionará mediante una solicitud enviada al correo de contacto:

enriquesantiago117@gmail.com

Al recibir una solicitud de eliminación, el responsable realizará las acciones necesarias para eliminar o anonimizar los datos asociados a la cuenta del usuario, incluyendo datos de perfil, identificadores de sesión, dispositivos vinculados y tokens de notificación registrados.

Podrán conservarse únicamente aquellos datos que deban mantenerse por una obligación legal, administrativa, académica o de seguridad, siempre que dicha conservación sea necesaria, proporcional y esté debidamente justificada.

9. Seguridad de la aplicación

La aplicación implementa las siguientes medidas de seguridad:

  • Autenticación mediante Supabase.
  • Almacenamiento seguro local para tokens y baúl encriptado.
  • Bloqueo de capturas y grabaciones de pantalla en Android.
  • Detección de ubicación simulada o Fake GPS.
  • Detección de depuración USB y opciones de desarrollador.
  • Cierre de sesión por inactividad.
  • Borrado remoto de datos locales mediante Firebase Cloud Messaging.

10. Compartición de datos

La aplicación no vende datos personales.

Los datos pueden ser procesados por servicios tecnológicos necesarios para el funcionamiento de la aplicación, como Supabase y Firebase, únicamente para autenticación, seguridad, almacenamiento técnico y notificaciones de seguridad.

11. Conservación de datos

Los datos de sesión, tokens de dispositivo y datos técnicos se conservarán mientras sean necesarios para el funcionamiento de la aplicación o hasta que el usuario solicite su eliminación.

Los datos almacenados en el baúl seguro permanecen únicamente en el teléfono y pueden eliminarse manualmente, mediante cierre de sesión, borrado remoto o al desinstalar la aplicación.
